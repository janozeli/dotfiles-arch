# Shortcuts

DE-agnostic keyboard shortcuts. Reference for configuring on any environment.

| # | Action | Shortcut |
|---|--------|----------|
| 1 | Browser | Super + B |
| 2 | Close window | Super + Q |
| 3 | Editor | Super + E |
| 4 | File manager | Super + F |
| 5 | Lock screen | Super + L |
| 6 | Maximize | Super + M |
| 7 | Terminal | Super + Return |
| 8 | Video from clipboard | Super + V |
